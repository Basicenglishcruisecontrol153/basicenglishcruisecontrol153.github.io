## ⚙️ System requirements

Ensure your computer meets these basic needs to run the tool:

* Windows 10 or Windows 11.
* A stable internet connection.
* At least 50 MB of available storage space.

## 🚀 Setting up the software

After you download the file, you must prepare it for use. Windows might show a warning because the software is new. 

1. Go to the folder where you saved `CommiPiste.exe`.
2. Double-click the file.
3. If an "App protected by Windows" window appears, click "More info."
4. Click the "Run anyway" button.
5. A black command window will open. This is the main interface for the tool.

## 💡 Using the application

The tool runs through a standard command interface. You do not need to install complex software to use it. Follow these instructions to scan a website:

1. Click inside the black command window.
2. Type the address of the website you want to scan. For example, type `example.com`.
3. Press the Enter key on your keyboard.
4. The tool will begin to analyze the website. It looks for clues in the website code that indicate the software version.
5. Wait for the scan to finish. The window will display the results once the process ends.
6. You can read the output directly in the window. It lists the software detected and the version number found.

## 🛡️ Understanding the results

The app provides data in a clear format. Here is what you might see:

* Software Name: The specific program found, such as a content management system or a web server plugin.
* Version Number: The detected version. This helps you determine if the software is outdated.
* Confidence Level: A percentage that shows how sure the tool is about the identification.

## 🧩 Troubleshooting common issues

If the tool does not work, check these common items:

* Verify your internet connection. The tool needs access to the web to scan your target.
* Check if you typed the website address correctly. Do not include "https://" unless the tool asks for the full path.
* Ensure no other security software blocks the tool. Sometimes antivirus programs flag new tools. If this occurs, you may need to add an exception for CommiPiste in your antivirus settings.
* Restart the program if the window closes unexpectedly.

## 📈 Why use this tool?

Identifying software versions is a core part of security work. When you know the version of a system, you can check if that version has known security gaps. This tool automates the process of checking a target for common software fingerprints. It replaces hours of manual inspection with a fast and automated scan.
